Клееный брус

Процесс изготовления клееного бруса

1. Сортировка древесины

Для производства клееного бруса мы используем только качественный материал, который отбирается по ряду строгих параметров. Мы используем только местное сырье — северный лес. Пример ссылки. Соответственно древесина имеет повышенные прочностные свойства, а высокое содержание смолы делает материал более долговечным.

2. Сушка

Как ни странно, для непосвященного человека, один из самых сложных в технологическом плане процессов — процесс сушки. От влажности древесины зависят следующие характеристики клееного бруса:

  • прочность клеевых соединений;
  • стойкость от поражения грибком;
  • усыхание и появление трещин;
  • коробление бруса;
  • чистота обработки поверхностей.

Для сушки древесины на нашем предприятии применяются управляемые электроникой конвективные камеры, которые обеспечивают хорошие показатели по распределению влажности, и снятию анизотропных напряжений доски. Для изготовления клееного бруса мы применяем доску влажностью только 10-12%.

Важно знать что эксплуатационная влажность (влажность древесины при которой она будет эксплуатироваться) в деревянных домах может достигать 5%. Соответственно усыхание стены дома из клееного бруса шириной, допустим, 200 мм с влажности 18% по ГОСТу составляет 3,6 мм!

3. Изготовление заготовок

Изготовление заготовок. Этот процесс состоит из двух этапов. Сначала высушенная древесина направляется на станки, которые аккуратно вырезают из неё все дефекты, поврежденные участки. После чего она нарезается на ламели, которые сращиваются по длине.

4. Склеивание

От того какой клей был использован для производства бруса, будет зависеть его прочность и экологичность. На нашем предприятии для склейки бруса используется ЭПИ клей всемирно известного производителя фирмы «Акго Мобеь. один факт, что она существует более 100 лет, дает гарантию высоких потребительских свойств. Это подтверждается многочисленными лабораторными испытаниями

Многие производители, идя на сокращение издержек при производстве бруса, отказываются от склейки по длине. В итоге доска с внутренними напряжениями иналичием недопустимых дефектов на лицевой стороне оказывается в брусе. Мы сознательно применяем склейку по длине, которая снимает внутренние напряжения и позволяет использовать только эстетично-выглядящие участки дерева на наружных поверхностях бруса. Для изготовления бруса высотой 240 и 285 мм, всё применяется технология склейки бруса из двух полубрусьев или «блочная склейка». В данном случае сосну применять не желательно, из-за разности в цвете.

5. Профилирование

Еще один важный аспект в производстве клееного бруса - это точность станков на которых производится раскрой биндера (погонажной заготовки). В наше время используется большое количество марок и видов станков для нарезки чаш, сверлений изготовления пазов. Многочисленные станки отечественного и китайского производства во многом копируют европейских производителей, но не дают такого постоянства геометрии элементов бруса. Применяемые на нашем комбинате станки с ЧПУ фирм Нипаеддег(Германия) и КгизК Швейцария) позволяют воплотить самые смелые фантазии инженеров в дереве, с непревзойденной точностью.

При изготовлении чашек допуск расположения составляет +/-1мм. Отклонение размеров самой чашки +/0.25 мм. Наш 7-образный тепловой замок (2-1 0сК) в чашке исключает прямое продувание и значительно уменьшает тепловые потери. При подгонке размеров чашки мы проверяли теплопроводность данного узла с помощью тепловизора.
